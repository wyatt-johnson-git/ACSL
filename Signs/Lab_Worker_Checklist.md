# Lab Worker Daily Checklist

## OPEN LAB
- [ ] Unlock doors
  - [ ] Main door
  - [ ] Podcast studio
  - [ ] Server room
- [ ] Turn on lights
- [ ] Turn on sign-in monitor

## DURING THE DAY
- [ ] Set door contact sign out (5:30-6:00pm)

## CLOSE LAB
- [ ] Retrieve door sign (7:30-8:00pm)
- [ ] Push in chairs
- [ ] (Monday, Teusday, Wednesday) set robovac schedule for 7am
- [ ] Shut & lock all doors
  - [ ] Back door
  - [ ] Podcast studio
  - [ ] Server room
- [ ] Turn off sign-in monitor
- [ ] Turn off lights
- [ ] Shut & lock main door
